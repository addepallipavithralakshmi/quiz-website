# Interactive Quiz App

An interactive and responsive Quiz Application built using **HTML, CSS, and JavaScript**.  
This project allows users to select quiz topics, answer multiple-choice questions, track scores, and view results with a timer feature.

---

## 🚀 Features

- Multiple quiz categories
- Responsive user interface
- Timer for each quiz
- Score calculation
- Leaderboard functionality
- Multiple-choice questions
- Beautiful card-based topic selection
- Question numbering
- Instant result display

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
